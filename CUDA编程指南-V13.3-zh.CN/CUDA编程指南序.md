# CUDA 编程指南

## CUDA 和 CUDA 编程指南
CUDA是由 NVIDIA 开发的一个并行计算平台和编程模型，利用GPU的强大能力显著提升计算性能。它使得开发者可以加速计算密集型应用，并广泛应用于像深度学习、科学计算及高性能计算等领域。

本CUDA 编程指南是关于CUDA编程模型及如何使用CUDA平台编写在GPU上执行的代码的官方综合资源。本指南涵盖了从CUDA编程模型和CUDA平台到语言扩展细节的方方面面，并且介绍了如何利用特定硬件和软件特性。本指南为初学开发者提供了一个学习CUDA的途径，同时也为使用CUDA创建应用程序的开发者提供了一个必不可少的资源。

## 本指南的组织结构
即使对主要使用库、框架、或者领域特定语言的开发者来说，理解CUDA编程模型以及GPU如何执行代码也是有价值的，有助于了解抽象层背后发生了什么。本指南首先用一章的篇幅介绍CUDA编程模型，它与任何特定编程语言无关，适用于任何对CUDA工作原理感兴趣的人，甚至是非开发者。

本指南分为以下五个主要部分：
* 1. 引言和编程模型摘要
    * 语言无关的CUDA编程模型概述和CUDA平台的简短介绍。
    * 本节内容面向所有希望理解GPU以及在GPU上执行代码概念的人，甚至是非开发者。

* 2. 使用CUDA对GPU进行编程
    * 使用CUDA C++ 进行GPU编程的基础知识。
    * 本节内容面向所有想要入门GPU编程的人。
    * 本节主旨是提供指导，而不是介绍全部内容，主要讲解CUDA编程中最重要和最常见的部分，包括一些常见的性能考量。

* 3. 高级CUDA
    * 介绍一些CUDA的高级特性，这些特性既能够提供更精细的控制又能提供更多提升性能的机会，包括在单个应用程序中使用多个GPU。
    * 本节最后列举出第四部分的各项功能，简要介绍每种功能的用途，并按开发者可能使用每项功能的时机和原因排序。

* 4. CUDA 特性
    * 本节全面介绍了CUDA的特定特性，如CUDA计算图、动态并行、与图形API的互操作性及统一内存。
    * 当需要详细了解某个特定CUDA特性时可以参考本节内容。前面的章节可能也介绍并阐述了本节涵盖的特性。

* 5. 技术附录
    * 技术附录提供了一些关于CUDA的C++高级语言支持、硬件特定规范和其他技术规范的参考文档
    * 本节旨在作为具体描述CUDA元素的语法、语义、和技术行为的技术参考。

1-3节为CUDA初学开发者提供了学习经验指南，同时也为任何有CUDA经验的开发者提供了有用的见解和最新的信息

第4和第5节提供了关于特定特性和详细主题的丰富信息，旨在为开发CUDA应用时需要了解更多细节的开发者提供精心整理、组织良好的参考文档。
